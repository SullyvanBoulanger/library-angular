## Installation

1. Clone this project

```sh
git clone git@github.com:SullyvanBoulanger/library-angular.git
```

2. Install dependencies

```sh
# npm
npm i
# yarn
yarn i
# pnpm
pnpm i
```

## Launch

1. Run json-server in a terminal

```sh
npm run json-server
```

2. Run angular app in another terminal

```sh
ng serve -o
```
